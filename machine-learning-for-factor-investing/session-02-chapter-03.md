# Session 02 — Chapter 3：因子投资与资产定价异常

阅读范围：Chapter 3 `Factor investing and asset pricing anomalies`  
PDF 页码：约 p32–55  
定位：这是进入模型章节前最重要的金融底座。先理解“要预测什么”和“为什么这些特征可能有用”，再谈树模型、神经网络、SVM。

## 1. 本章一句话

本章讲的是：

> 因子投资试图用股票的共同特征或共同风险暴露解释横截面收益差异；所谓“异常”，就是传统模型难以解释、但在历史数据中看起来能带来超额收益的规律。

机器学习在后面接入的位置是：

```text
传统因子投资：少数人工定义特征 -> 排序/回归 -> 组合
机器学习因子投资：大量公司特征 -> 非线性模型 -> 预测收益/构造组合
```

但二者共同面对同一个核心问题：

> 哪些特征真的含有关于未来收益的信息？

## 2. 为什么要先读第 3 章

如果直接跳到模型部分，很容易误以为任务是：

> 给我一个 `X` 和 `y`，训练一个模型，比较 RMSE/accuracy。

但在 factor investing 里，真正的问题是：

1. `X` 的每一列是否有经济含义？
2. `y` 是未来收益、超额收益、分组标签，还是组合权重目标？
3. 一个统计显著的信号是否只是数据挖掘？
4. 一个预测信号扣掉交易成本后还赚钱吗？
5. 这个信号是否已经被市场交易掉了？

因此第 3 章是“模型部分”的前置语境。

## 3. 核心概念：资产定价异常

“异常”不是说市场出 bug，而是指：

- 某些股票特征与未来收益存在稳定关系；
- 这种关系不能被经典资产定价模型充分解释；
- 例如小市值、价值、动量、高盈利、低投资、低风险等。

常见因子：

| 因子 | 英文/缩写 | 直觉 |
|---|---|---|
| Size | SMB, small minus big | 小公司长期可能获得更高收益 |
| Value | HML, high minus low | 便宜股票相对贵股票可能有溢价 |
| Momentum | WML, winners minus losers | 过去赢家短中期继续赢 |
| Profitability | RMW, robust minus weak | 高盈利公司表现更好 |
| Investment | CMA, conservative minus aggressive | 低投资/稳健扩张公司表现更好 |
| Low risk | BAB 或 low volatility/beta | 低风险股票可能有反常溢价 |

读模型章节时要记住：这些因子不是“特征工程玩具”，而是金融解释与投资假设。

## 4. 检测异常的三条主线

### 4.1 Portfolio sorting：排序组合法

最直观的方法：

```text
每个月：
1. 按某个特征给股票排序，例如市值、估值、动量；
2. 分成若干组，例如十分位；
3. 买一端、卖另一端，或比较两端收益；
4. 得到一条 long-short 因子收益序列；
5. 检查均值、t-stat、夏普、稳定性、换手和成本。
```

优点：直观、贴近组合构造。  
缺点：一次通常只看少数特征，容易漏掉非单调关系和特征交互。

### 4.2 Factor model：因子模型

书中给出线性因子模型的基本形式：

```text
r_{t,n} = alpha_n + sum_k beta_{n,k} f_{t,k} + epsilon_{t,n}
```

含义：一只股票的收益可以由若干共同因子的暴露解释。

这里的关键不是公式，而是思维方式：

- `f_k` 是共同风险/收益来源；
- `beta` 是股票对这些来源的暴露；
- `alpha` 是模型解释不了的剩余收益；
- 如果某个策略长期有正 alpha，就可能是异常，也可能是模型遗漏风险、数据挖掘或成本没算全。

### 4.3 Fama-MacBeth 回归

这是横截面资产定价常用方法。粗略理解：

```text
每个时间点 t：
    用当期所有股票做一次横截面回归：未来收益 ~ 当前特征
跨时间：
    看每个特征系数的平均值和显著性
```

它比简单排序更适合同时检验多个特征，但仍然通常是线性框架。

机器学习模型后面要做的事，可以看成把这里的线性关系扩展为：

```text
未来收益 = f(大量特征)
```

其中 `f` 可以是树、boosting、神经网络、SVM、贝叶斯模型等。

## 5. 从传统因子到机器学习的连接

本章最后部分很关键：作者把资产定价和机器学习接起来。

传统方法通常是：

```text
少数特征 + 线性模型 + 经济解释
```

机器学习方法倾向于：

```text
大量特征 + 非线性/交互 + 预测性能
```

但机器学习不能绕开资产定价问题。它只是换了函数形式：

```text
传统：r = alpha + beta * factor + error
ML：r = f(characteristics) + error
```

更高级的连接包括：

- 用特征预测因子暴露；
- 用神经网络生成非线性 beta；
- 用 autoencoder 从收益中学习潜在因子；
- 把公司特征网络和收益 autoencoder 结合起来。

这解释了为什么后面神经网络、autoencoder、PCA、无监督学习不是孤立技巧，而是用于学习“隐藏因子结构”。

## 6. 本章最重要的警告

### 6.1 异常可能是假的

原因包括：

- publication bias：容易发表正结果；
- p-hacking：试太多变量，总会找到显著的；
- multiple testing：多重检验问题；
- 数据质量问题；
- 未计入交易成本；
- 样本外失效。

### 6.2 异常可能会衰减

一个因子公开之后，资金涌入会压低未来收益，这叫 alpha decay。

所以实战时要问：

- 这个信号是否过于拥挤？
- 它在发表后是否仍有效？
- 换手率和交易成本是否吃掉收益？
- 不同市场、不同时间段是否稳健？

### 6.3 经济解释仍然重要

纯数据挖掘找到的信号很脆弱。一个可用因子最好至少有一种解释：

- 风险补偿：承担某种系统性风险；
- 行为偏差：投资者长期犯类似错误；
- 市场结构：约束、摩擦、机构限制带来的错价。

## 7. 给“模型部分”的读法提醒

从第 5 章开始，每学一个模型都按下面框架读：

1. 它假设特征和收益之间是什么关系？线性、非线性、局部、分段、核空间？
2. 它如何防止过拟合？正则化、剪枝、bagging、boosting、dropout、先验？
3. 它能否处理特征交互？
4. 它适不适合金融低信噪比数据？
5. 它输出的是收益预测、分类标签，还是排序信号？
6. 它最终如何转成组合权重？

## 8. 自测题

1. 什么是资产定价异常？它和“套利机会”有什么区别？
2. 为什么 portfolio sorting 很直观但不够强？
3. Fama-MacBeth 回归在检验什么？
4. 为什么一个高 t-stat 的因子也可能不可交易？
5. 机器学习相对传统因子模型主要扩展了什么？
6. 为什么 alpha decay 对因子投资尤其重要？
7. 如果 XGBoost 找到了一个强预测特征，你会如何判断它是不是数据挖掘？

## 9. 本章到下一章的桥

第 3 章告诉我们：我们要从公司特征里找未来收益信号。  
第 4 章会告诉我们：在把特征喂给模型前，必须处理缺失值、极端值、标准化、标签定义、样本持久性等问题。

如果要读“模型部分”，我建议下一步不是直接第 5 章，而是先用第 4 章把数据预处理读掉。否则模型代码看起来能跑，但会不清楚哪些地方可能造成前视偏差或过拟合。
