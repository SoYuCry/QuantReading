## 2. 这本书的边界

Preface 里作者把范围限定得很清楚：

- 聚焦 **equity investment strategies**，也就是股票投资策略；
- 策略依赖 **firm characteristics**，例如市值、估值、盈利、资产增长、波动率、动量等；
- 目标是把机器学习工具用于 **factor investing**，而不是泛泛覆盖所有金融机器学习场景；
- 不重点讲高频交易、订单簿、NLP 情绪、欺诈检测、信用评分等。

一句话概括：

> 这本书研究的是：如何用公司特征和机器学习方法，构造、验证并解释股票因子投资策略。

## 3. Chapter 1：记号和数据怎么理解

书里会并行使用两套语言：

### 机器学习语言

- `X`：特征矩阵
- `y`：标签/预测目标
- 一行样本：一个 observation / instance
- 一列特征：一个 attribute / predictor / feature

### 金融语言

- `t`：时间
- `n`：资产/股票
- `r_{t,n}`：股票 `n` 在时间 `t` 的收益率
- `x^{(k)}_{t,n}`：股票 `n` 在时间 `t` 的第 `k` 个特征

关键桥接：

> 机器学习里的一个样本，通常对应金融里的一个 `(t, n)`，也就是“某个月的某只股票”。

这点非常重要，因为它决定了训练集、验证集、回测切分都必须尊重时间顺序，不能把未来信息泄漏到过去。

## 4. Chapter 2：为什么机器学习不是魔法棒

作者在引言里强调三件事：

### 4.1 金融 ML 的机会来自三件事

1. 数据更容易获得；
2. 计算能力更强；
3. 因子投资已有经济学基础。

### 4.2 但金融数据极难

主要难点：

- 信号弱，噪声强；
- 时间长度有限，尤其财务数据多为季度/月度；
- 市场会适应，简单有效规律可能被套利消灭；
- 相关性不等于因果；
- 非平稳性很强，过去有效不保证未来有效。

### 4.3 输入往往比模型更重要

作者的核心提醒可以概括为：

> 在因子投资里，特征、标签、样本切分、回测协议，通常比“选哪个高级模型”更重要。

所以读这本书时，不要陷入“XGBoost vs 神经网络”的模型崇拜，而要持续问：

- 标签定义是否合理？
- 特征是否有经济含义？
- 是否有前视偏差？
- 是否外样本有效？
- 是否考虑交易成本和换手？

## 5. 本次核心图景

一个 ML factor investing 工作流可以理解为：

```text
公司特征数据 X
    ↓
清洗 / 标准化 / 缺失值处理 / winsorization
    ↓
定义预测目标 y，例如未来一个月收益、分组标签、超额收益
    ↓
训练模型 f(X) -> y_hat
    ↓
把预测信号转成组合权重
    ↓
回测：收益、风险、换手、成本、因子暴露
    ↓
解释模型与检查稳健性
```
